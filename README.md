# Car Information in C

My first C programming project.

This project stores and displays basic information about a car and calculates its driving range based on fuel capacity and fuel consumption.

## Features

- Store car brand and model
- Store year and maximum speed
- Calculate fuel range
- Practice C data types
- Practice variables and operators
- Practice printf formatting

## Formula

Range = (Fuel Tank / Fuel Consumption) × 100
